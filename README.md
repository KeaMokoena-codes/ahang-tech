# AhangTech 

**Bringing Harmony and Precision to Construction Technology (ConTech).**

AhangTech is a modular Python-based suite designed to bridge the gap between physical construction sites and digital management. By unifying asset tracking, site reporting, and material estimation, AhangTech eliminates data fragmentation and improves site accountability.

---

## The Vision
In many construction environments, information is lost between paper logs and verbal check-ins. **AhangTech** (derived from the word for "Harmony") provides a centralized digital workflow for site managers and contractors.

### Core Modules
1.  **Asset Manager (Current Focus):** A digital chain of custody for high-value tools and equipment.
2.  **The Reporter (Upcoming):** Automated daily site logs including weather integration and incident reporting.
3.  **The Estimator (Upcoming):** A logic-heavy calculator for material procurement that accounts for environmental waste factors.

---

## Tech Stack
* **Language:** Python 3.x
* **Core Libraries:** 
    * `argparse` for professional CLI command handling.
    * `json` for persistent data storage.
    * `datetime` for precise tracking and logging.
* **Architecture:** Modular design for scalability.

---

## Asset Manager Features
* **Check-in/Check-out:** Link equipment to specific personnel with unique IDs.
* **Status Monitoring:** Real-time visibility on tool availability and repair needs.
* **Persistence:** All site data is saved to local storage—no data lost on shutdown.
* **Validation:** Robust error handling to prevent duplicate IDs or invalid user entries.
